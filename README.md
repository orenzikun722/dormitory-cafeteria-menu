## 仕組み
1. WindowsやLinuxでPythonを使い表形式のpdfからデータを抜き取る  
2. 抜き取ったデータをGoogleのサービスアカウントを使いスプレッドシートに書き込む
3. Google Apps Scriptを使いLINEからリクエストを受信する
4. そのリクエスト通りにスプレッドシートからデータを取得する
5. LINEの応答メッセージエンドポイントを使用しデータを送信する

表形式で作られていることを大前提としているため表が崩れたり文字のみだったりの場合正しくデータ化できない  
例えば長期休みのはじめで朝食だけの時、昼食と夕食のセルが一つに合体していると昼食のデータとして出してきたりしてエラーが出る  

## 改善点と問題点
- ***署名確認ができない*** <=ちょい重要
- pdfからPythonを使ってデータを抜き取る作業は人力(ファイル選ぶだけ)
- Google Apps Scriptを使っているため応答が少し遅い
- Google サービスアカウントの登録、設定が必要
- PythonはGoogle Colabで動かすことによって人間による作業を簡略化する
- 定期配信の時に料理の写真を出す(これは後回し)
- 朝に行事予定の配信
